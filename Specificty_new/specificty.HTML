Css specificy

css specificy อธิบายได้สั้นๆคือ css rules 2 ตัวที่มี selector ต่างกัน มาใช้ html ที่มี element เดียวกัน และโครงสร้างของ index.html และ home.html จะมีความคล้ายคลึงกันมากสีของปุ่ม home.html และ index.html จะมีข้อแตกต่างกับตัวเกิดจาการที่ link ของลำดับ style1 และ style2 สังเกตการทำงานของ style ได้ดังหัวข้อต่อไปนี้

Home

ภาพโครงสร้างของ home.html

จากภาพของตัวอย่างของ home.html ด้านบนสีของปุ่ม home และ index จะมีข้อแตกต่างกันซึ่งเกิดจาการที่ link ของลำดับในการเรียกตัว style-css1 และ style-css2 มีลำดับดังรูป

link ของ home.html จะมีการเชื่อมโยงของ external css style-2 แล้วตามด้วย style-1

    #style-2 จะพบว่ามี ul li.active a, ul li a, ul li.active เป็น selector

  • ul li.active a ต้องการให้คลาส active ใน tag a เป็นสี gold ซึ่งสามารถเทียบกับโครงาร้างของ code home.html ด้านบนได้
  • ul li a ต้องการให้คำว่า index แสดงเป็นสี red
  • li.active ต้องการให้พื้นหลังหน้าปุ่ม home เป็น transparent(พื้นหลังโปร่งใส)

    #style-1 ที่ทำงานต่อจาก style-2 กฎของcss rule 2 จะแสดงผลของ selector ของลำดับสุดท้ายเสมอ

  • ul li a ปุ่ม index จะแสดงสีแดง
  • ul li.active จะแสดงพื้นหลังเป็นสีเขียวเนื่องจากเอา selector ลำดับสุดท้าย

Index

ภาพโครงสร้างของ code index.html

จากภาพตัวอย่าง index.html ด้านบน สีของปุ่ม home และ index จะมีข้อแตกต่างกันเกิดจาการที่ link ของลำดับในการเรียกตัว stlye-css1 และ style-css2 มีลำดับดังรูป

    #style-1 จะเห็นได้ว่ามี selector ul li a, ul li.active

  • ul li a จะทำให้ home แสดงเป็น red ซึ่งสามารถเทียบกับโครงาร้างของ code index.html ด้านบนได้
  • ul li.active จะให้ ปุ่มindex กำสีเป็น gold

style-2 ที่ต่อจาก style-1 จะเห็นว่ามี selector ul li active a ทำให้ index ออกมาเป็น gold และ ul li a จะทำให้ home เป็น red เหมือนเดิมเพราะสีเหมือนกันส่วน ul li.active จะทำให้พื้นหลังโปร่งใส (ไม่มีสีพื้นหลัง)แทนที่สีเขียวดังรูปด้านบน

Summary

อย่างที่ทราบ จากหัวข้อ home และ index ข้างต้น จะเห็นได้ว่าสีของปุ่มสามารถเปลี่ยนแปลงได้ขึ้นกับลำดับของ css rule ซึ่งจะแสดงผลจาก selector ลำดับสุดท้าย
